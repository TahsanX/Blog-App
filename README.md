# Next.js Blog Application ‚Äì SEO-Optimized Personal Blogging Platform

This project is a fully responsive, SEO-optimized personal blog built with **Next.js**, **Tailwind CSS**, and **Velite.js** for content management. The application is designed with performance, scalability, and clean UI/UX best practices in mind. It supports dynamic routing, MD/MDX-based blog posts, syntax highlighting, reusable components, and smooth page animations.

- Dynamic blog post generation

- Code syntax highlighting for technical articles  
- Reusable UI components (author card, tags, blog card, layout sections)
